# 📱 Lost Phone Helper

Lost Phone Helper is a simple web application designed to help increase the chances of recovering a lost phone. It allows a person who finds a lost device to easily share their location and contact information with the owner.

When the application is opened, the user enters their email address and clicks the "Share Location" button. The browser then requests permission to access the device’s location. Once granted, the application captures the current GPS coordinates and generates a Google Maps link.

This link can be opened to view the exact location of the device, helping the owner identify where the phone is currently located.

---

## 🎯 Purpose of the Project

The main goal of this project is to demonstrate how basic web technologies can solve real-world problems while helping beginners learn:

- How to build interactive web applications  
- How to use browser APIs (like geolocation)  
- How to handle user input and events  
- How to deploy websites online using GitHub Pages  

---

## 🌍 Key Idea

Instead of using complex tracking systems, this project focuses on a simple and practical approach where cooperation between the finder and the owner helps in recovering lost devices.

## 🔐 Privacy Note

The application only requests location access when the user clicks the button and gives permission. No data is stored or shared automatically.
